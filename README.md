# 🎓 Online Course Landing Page

A modern, high-converting **education landing page** built for selling online courses with a clean interface, responsive layout, coupon-driven promotions, and a conversion-focused popup experience.

This project is developed using **HTML, CSS, and JavaScript** and is designed to be simple, scalable, and marketing-friendly for course promotions, digital products, and education-based campaigns.

## ✨ Features

- 📱 Fully responsive design (Mobile, Tablet, Desktop)
- 🎓 Professional education landing page layout
- 🧠 Course-focused content structure with modern UI
- 🎁 Coupon section with copy functionality
- 🔥 Full-screen popup (auto trigger after 10 seconds)
- 🚀 Smooth scrolling navigation
- 💚 Clean UI (white background, black text, light green buttons)
- ⚡ Fast performance (no frameworks used)
- 🎯 Conversion-focused structure
- 🔗 Global CTA link control system

---

## 💡 Popup Experience

The project includes a **full-screen popup system** that:

- Automatically opens after 10 seconds  
- Showcases course highlights and categories  
- Promotes offers and learning benefits  
- Uses a **Continue button** to guide user flow  
- Enhances user engagement and conversions  

---

## 🧠 Global Button Link Control

All CTA buttons can be controlled using a **single link variable** in `script.js`.

Example:

    const MAIN_LINK = "https://your-link.com";

Use cases:

- Affiliate marketing (CPA / CPI)
- Course selling funnels
- Paid ad landing pages
- Fast link switching for campaigns
